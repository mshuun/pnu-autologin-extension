# 🎓 PNU Auto Login (부산대학교 자동 로그인)

**PLATO** 및 **학생지원시스템** 자동 로그인 크롬 익스텐션입니다.

## 주요 기능 (Features)
*   **자동 로그인:** PLATO 및 Onestop 접속 시 저장된 계정 정보로 자동 로그인합니다.

## 설치 방법 (Installation)
**개발자 모드**를 통해 설치할 수 있습니다.

1. 이 리포지토리를 클론(`git clone`)하거나 ZIP 파일로 다운로드하여 압축을 풉니다.
2. 크롬 브라우저 주소창에 `chrome://extensions/`를 입력하여 접속합니다.
3. 우측 상단의 **'개발자 모드'** 토글을 켭니다.
4. 좌측 상단의 **'압축해제된 확장 프로그램을 로드'** 버튼을 클릭합니다.
5. 다운로드한 폴더를 선택하면 설치가 완료됩니다

## 사용 방법
1. 익스텐션 설치 직후 자동으로 **설정 페이지**가 열립니다.
2. 부산대학교 아이디(햑번)와 비밀번호를 입력하고 '저장하기'를 누릅니다.
3. 이제 PLATO 또는 학생지원시스템에 접속하면 자동으로 로그인이 진행됩니다.

## 보안 및 프라이버시 (Security & Privacy)
이 익스텐션은 외부 서버와 통신하지 않습니다. 사용자의 계정 정보는 오직 사용자 본인의 기기 내부(Chrome Local Storage)에만 암호화된 상태로 보관됩니다.
